# PrepPeer

AI mock interviews with real peer ranking — Next.js 14 frontend (mock data).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Optimized for [Vercel](https://vercel.com): import this repo and deploy. No environment variables required for the demo.

## Stack

- Next.js 14 (App Router)
- TypeScript, Tailwind CSS
- Framer Motion, Lenis, Recharts, Lucide
